## whj_ui
## 使用 React+typescript 开发

### 安装最后已经发布的组件库

~~~javascript
npm install whj_ui --save
~~~

### 使用

~~~javascript
// 加载样式
import 'whj_ui/dist/index.css'
// 引入组件
import { Button } from 'whj_ui'
~~~
